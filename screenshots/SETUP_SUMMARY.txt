8. TROUBLESHOOTING QUICK REFERENCE
   
   Issue: Connection refused to localhost:3000
   Solution: Start the frontend server (see step 1 above)
   
   Issue: Browser not found
   Solution: Use advanced_screenshot.js which falls back to HTML export
            Or install chromium: apt-get install chromium-browser
   
   Issue: Screenshots are HTML files, not PNG
   Solution: Convert HTML to images:
             - Use browser: firefox --headless --screenshot
             - Use tool: wkhtmltoimage file.html file.png
             - Use conversion: npx html2pdf
